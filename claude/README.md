# CuZn — Modern Heritage

A premium single-page website built around the concept of **Modern Heritage**: where deep craft tradition meets contemporary design clarity.

## Design Concept

- **Strong typography** — Playfair Display serif paired with Inter sans-serif for a classic-modern contrast
- **Large whitespace** — Generous spacing (`8rem`–`12rem` section padding) lets content breathe
- **Subtle scroll motion** — Intersection Observer-driven reveal animations and parallax on the hero
- **Brass details** — Warm ochre/gold accent colour (`#b08d57`) used sparingly for rules, labels, and hover states

## Structure

| Section | Description |
|---------|-------------|
| Hero | Full-viewport title with scroll indicator |
| Philosophy | Two-column editorial layout |
| Craft | Three-card grid detailing materials, process, and precision |
| Statement | Dark full-width blockquote |
| Collection | Two-column product grid with hover effects |
| Contact | Split-layout form |
| Footer | Minimal brand footer |

## Files

- `index.html` — Semantic markup, no framework dependencies
- `styles.css` — Custom properties, CSS Grid, responsive breakpoints
- `script.js` — Scroll reveal, parallax, nav state, form handling

## Usage

Open `index.html` in a browser. No build step required.

Replace placeholder collection items with product photography and update copy to match your brand.
