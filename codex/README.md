CuZn is a premium single-page concept for Trade Etching Direct focused on luxury brass signage for hotels and private residences.

Files:
- `index.html`: Editorial landing page with hero, craftsmanship, sectors, and CTA.
- `styles.css`: Cinematic visual system with brass-led palette, glass panels, fine borders, and responsive layout.
- `script.js`: Scroll reveal, magnetic hover response, and subtle tilt interactions.

Design direction:
- Minimal, high-contrast, materially rich.
- Serif-led typography for an editorial luxury feel.
- Motion kept restrained and tactile rather than flashy.

Usage:
Open `index.html` in a browser. The page is fully static and requires no build step.
