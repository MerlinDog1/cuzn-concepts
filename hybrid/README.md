# CuZn Hybrid Concept

A hybrid direction that combines:
- **Claude concept visual language**: modern-heritage typography, editorial spacing, elegant section rhythm.
- **Codex concept copy quality**: stronger premium tone focused on architectural brass signage and luxury environments.

## Files
- `index.html` — single-page concept with hero, philosophy, craft, sectors, and contact CTA
- `styles.css` — responsive design system with restrained brass accents and subtle hover/scroll interactions
- `script.js` — reveal-on-scroll, sticky nav state, smooth anchor scrolling, and light hero parallax

## Notes
- Mobile responsive at `900px` breakpoint.
- Motion is intentionally subtle and respects `prefers-reduced-motion` where applicable.
- Fully static (no build tools required).
